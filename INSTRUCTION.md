## Running the Container
1. Pull the Docker image:
   ```bash
   docker pull mykhailoivchenko/todoapp:1.0.0
   ```

2. Run the container:
   ```bash
   docker run -p 8080:8080 mykhailoivchenko/todoapp:1.0.0
   ```
